## üöÄ About The Project

This repository hosts the source code for the **Stellar x402 Portal**, a comprehensive resource for developers building with the x402 payment protocol on Stellar.

The x402 protocol turns the dormant HTTP 402 Payment Required status code into a fully-featured, on-chain payment layer for APIs, websites, and autonomous agents.

## üèÅ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/mertcicekci0/stellar-x402-portal.git
   cd stellar-x402-portal
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Visit [http://localhost:3000](http://localhost:3000)
